# CrewMatch Roadmap

## Visión
CrewMatch será una app para comparar roles de pilotos, tripulantes, parejas, amigos y grupos, detectando cuándo coinciden libres.

## Versión estable actual
- Lectura de archivos .ics
- Modo pareja
- Días libres juntos
- Calendario básico
- GitHub Pages funcionando

## Próximas versiones

### v1.2.1
- Próximo día libre juntos
- Mejor escapada del mes
- Validación anti-NaN
- Mejor texto de resumen

### v1.3
- Tarjetas diarias en lugar de tabla
- Mejor diseño para iPhone
- Calendario más limpio

### v1.4
- Modo grupo completo
- Subir varios roles
- Detectar cuándo todos coinciden libres
- Detectar cuándo coinciden 3 de 4, 4 de 5, etc.

### v2.0
- Nueva arquitectura
- Código separado
- Diseño premium
- App instalable en iPhone
- Pernoctes inteligentes
- Coincidencia de ciudad
- Coincidencia de vuelos

## Funciones futuras
- Historial mensual
- Comparar meses
- Guardar nombres
- Exportar resultados
- Compartir con pareja o grupo
