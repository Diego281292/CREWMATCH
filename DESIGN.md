# CrewMatch Design

## Identidad
CrewMatch debe sentirse como una app premium para pilotos y tripulantes.

## Estilo visual
- Modo oscuro
- Azul profundo
- Verde para coincidencias buenas
- Amarillo para ventanas medias
- Rojo para días difíciles
- Tarjetas grandes y limpias
- Diseño cómodo para iPhone

## Pantallas principales

### Inicio
- Couple Score / Group Score
- Próximo momento juntos
- Próximo día libre
- Mejor escapada
- Botón para analizar nuevo rol

### Calendario
- Vista mensual
- Colores por tipo de día
- Tocar un día para ver detalle

### Detalle del día
- Estado de cada persona
- Horas libres en común
- Recomendación del día

### Estadísticas
- Horas juntos
- Días completos
- Noches útiles
- Mejor bloque
- Comparación mensual futura

### Ajustes
- Modo pareja / grupo
- Nombres
- Exportar
- Compartir
